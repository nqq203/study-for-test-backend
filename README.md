# SFT BACKEND APP 

## DESCRIPTION
Đã config xong một số thư viện cần thiết\
Đã xong các API liên quan đến Sign In, Sign Up, Log out\
Viết thêm các API - Repo - Service - Route cho việc đọc file PDF\

### Install dependencies
```bash
npm install 
```

### Run project
```bash
npm run dev 
```
